Approximation algorithms provide a way to find a solution that is guaranteed to be within a certain range or factor of the optimal solution.

Choose the target Function :

The program needs only to learn how to choose the best move from among these legal moves.

B -> M to indicate that this function accepts as input any board from the set of legal board states B (b1+b2+…..+bn) and produces as output some move from the set of legal moves M (m1+m2+…..+mn)

1.if b is a final board state that is won, then V(b) = 100
2.if b is a final board state that is lost, then V(b) = -100
3.if b is a final board state that is drawn, then V(b) = 0
4.if b is a not a final state in the game, then V(b) = V(b'), where b' is the best final board state that can be achieved starting from b and playing optimally until the end of the game (assuming the opponent plays optimally, as well).

Nonoperational definition : for a particular board state requires (case 4) searching ahead for the optimal line of play, all the way to the end of the game but is not efficiently computable so called as Nonoperational definition.

Representation of target Function :

x1: the number of black pieces on the board
x2: the number of red pieces on the board
x3: the number of black kings on the board
x4: the number of red kings on the board
x5: the number of black pieces threatened by red (i.e., which can be captured on red's next turn)
X6: the number of red pieces threatened by black

Thus, our learning program will represent Ṽ(b) as a linear function of the form

(b) = W0 + w1x1 + w2x2 + w3x3 + w4x4 + w5x5 + w6x6

Where wo through W6 are numerical coefficients/weights

Estimating training Value :

Let take the example of a relay race/checker game / Chess :

A relay race is a type of race where teams of runners take turns to complete a course, passing a baton or other object between teammates at the end of each leg. The goal is for the team to complete the course as quickly as possible, with each runner contributing to the overall time.

In a typical relay race, the course is divided into equal sections, or legs, of a certain distance. Each team member runs one leg of the race before passing the baton to the next runner. The order in which the team members run is predetermined and often depends on the strengths and abilities of each runner.

The Rule for estimating training value been represented as

Vtrain(b) <- v^(Successor(b))
