Formula :
1.For +ve samples , move from Specific to Generic
2. For –ve sample, move from Generic to Specific

a. Been consider only if the attributes value is changing and the +ve case value will replace the –ve attributes value
b. Do not consider at all in case there is no change in the attributes value
c. In this case the values has to be in < ?,xx,?,?,?,? > format.
